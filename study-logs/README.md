# Study Logs & Knowledge Base

앞으로 학습하는 내용과 프로젝트를 진행하며 마주칠 트러블슈팅 사례들을 체계적으로 기록하기 위한 저장소입니다.

## 📂 디렉토리 구조

- **[learning/](./learning)**: 새로운 기술, 개념, 강의 내용 등을 기록합니다.
- **[troubleshooting/](./troubleshooting)**: 에러 해결 과정, 디버깅 기록, 성능 최적화 사례 등을 기록합니다.
- **[setup/](./setup)**: 프로젝트 환경 초기 설정 및 배포 가이드를 기록합니다.
- **[assets/](./assets)**: 문서에 사용되는 이미지, 다이어그램 등의 자산을 저장합니다.

## ✍️ 기록 가이드

기록 시 파일명은 `YYYY-MM-DD-제목.md` 형식을 권장합니다. (예: `2026-01-18-react-router-basename.md`)

- 새로운 포스트를 작성할 때 필요한 이미지는 `assets/images/` 폴더에 넣고 상대 경로로 참조하세요.
- 각 문서 상단에는 핵심 요약을 작성하여 나중에 찾기 쉽게 관리하세요.
